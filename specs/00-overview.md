## 概要
デスク環境を充実させるアイテムの人気ランキングサイト。複数のデータソースから情報を収集し、ユーザーが最適なアイテムを見つけられるようにする。

## 背景・目的
- テックブックランク（techbookrank.com）のデスク環境アイテム版
- デスク環境構築に悩むユーザーの意思決定をサポート
- Amazon アフィリエイトによる収益化

## ターゲットユーザー
1. **リモートワーカー**: 自宅のデスク環境を改善したい人
2. **エンジニア/クリエイター**: 生産性を高めるツールを探している人
3. **ガジェット好き**: 最新のデスクアイテムに興味がある人

## データソース
| ソース | 収集方法 | 用途 |
|--------|---------|------|
| X/Twitter | API + スクレイピング | 話題性・トレンド |
| YouTube | YouTube Data API | レビュー評価・人気度 |
| Amazon | PA-API | 売れ筋・価格情報 |
| 手動 | 管理画面 | キュレーション |

## ランキングアルゴリズム（概要）
```
総合スコア = (SNS言及スコア × 0.3) + (YouTube評価スコア × 0.3)
           + (Amazon売れ筋スコア × 0.2) + (エディター評価 × 0.2)
```

## 非機能要件
- **パフォーマンス**: Lighthouse スコア 90+
- **SEO**: 構造化データ対応、OGP 設定
- **アクセシビリティ**: WCAG 2.1 AA 準拠
- **レスポンシブ**: モバイルファースト

## 制約・前提条件
- Amazon PA-API の利用規約を遵守
- X/Twitter API の rate limit を考慮
